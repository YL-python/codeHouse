### vue 的更新队列
```
vue 会将一个事件循环中的所有数据操作合并，然后在执行更新
并不是数据改变一次就更新一次
1 2 3 4 5 4 3 2 1  这样的只会更新一次
```

### vue.use()
```
自动执行引入的模块 内部的 install 方法
install 方法 一般是进行全局注册组件

vue.use() 就相当于是全局注册组件。一般是写组件给别人用的时候
```

### 双向数据绑定 v-model
```
自定义事件和自定义属性的结合
可以使用在 表单元素上  和组件上

组件上使用 v-model
提供一个 input 的自定义事件
提供一个 value 的自定义属性
```
