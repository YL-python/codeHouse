#### 爬虫的步骤
1.通过http 或者 https 模块 获取网络资源（网页 记事本 图片 音频 视频）
```
http.get(url,(res)=>{
  let rawData=''
  res.on('data',(chunk)=>{
    <!-- 传递完一段数据就会触发 chunk 传递的每一段数据 -->
    rawData+=chunk
  })

  res.on('end',()=>{
    <!-- 数据传输完毕 -->
  })
})
.on('error',()=>{
})
```
2.分析内容 （正则表达式）
