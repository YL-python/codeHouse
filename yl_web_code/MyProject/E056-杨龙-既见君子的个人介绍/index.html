Welcome to my Website

既见君子，云胡不喜。 是我从高中以来最喜欢的句子，所以既见君子就一直是我使用的网名。

我叫杨龙，我来自江西九江，现在就读于南昌理工学院。专业是大数据，一开始高考填报志愿的时候我也没想太多 ，感觉大数据很高大上就报了这个专业。现在我已经大二了，我十分热爱我的专业，爱计算机，爱编程，爱动漫。
